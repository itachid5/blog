User Install Commands

Replace PUBLIC_URL with this dashboard's public URL and USER_TOKEN with the full token shown immediately after creation.

Install

Linux SSH

curl -fsSL -H "bypass-tunnel-reminder: true" https://PUBLIC_URL/install.sh | bash
nekotunnel token USER_TOKEN --api https://PUBLIC_URL

Windows RDP

iwr -UseBasicParsing -Headers @{"bypass-tunnel-reminder"="true"} https://PUBLIC_URL/install.ps1 | iex
nekotunnel token USER_TOKEN --api https://PUBLIC_URL

Temporary foreground mode

Linux

nekotunnel tcp 22

Connect:

ssh username@SERVER_ADDR -p SERVER_PORT

Windows

nekotunnel tcp 3389

Connect:

mstsc /v:SERVER_ADDR:SERVER_PORT

On Windows, Remote Desktop must be enabled and allowed in Firewall. If RDP is not enabled, nekotunnel can connect but RDP will not respond.

Background mode

Linux

nekotunnel start tcp 22
nekotunnel status
nekotunnel logs
nekotunnel stop

Windows

nekotunnel start tcp 3389
nekotunnel status
nekotunnel logs
nekotunnel stop

Backward-compatible Linux one-shot

curl -fsSL -H "bypass-tunnel-reminder: true" https://PUBLIC_URL/install.sh | bash -s -- tcp 22 USER_TOKEN https://PUBLIC_URL

The client downloads frpc v0.57.0 automatically from this NekoTunnel Central server if it is missing.
